# research/ — Evaluation & Benchmarks

Evaluation harnesses and benchmark data, primarily for `council/` (does multi-agent consensus actually beat a single-agent baseline? this is the most important open question in the project) and, later, `native/aegis/` (isolation test suite) and `native/lunar-core/` (boot/regression benchmarks).

Doubles as the data source for the companion research paper — see `docs/PRD.md` Appendix B for how these map to a paper's Experimental Evaluation section.
